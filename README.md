# Product Operations Playbooks

This repository contains practical playbooks to help Product Managers improve execution efficiency across the product development lifecycle, starting with AI-assisted Shape Up pitching.

## Contents

- `docs/product-ops-framework.md`  
  End-to-end Product Operations framework (lifecycle, cadence, roles, KPIs).
- `docs/shape-up/ai-pitch-playbook.md`  
  AI-assisted operating workflow for creating high-quality Shape Up pitches.
- `docs/shape-up/templates/shape-up-pitch-template-ai-overlay.md`  
  Reusable pitch template that maps to current sections and adds AI guidance.

## How to use

1. Review the framework and agree on lifecycle cadences.
2. Run the AI pitch workflow during shaping.
3. Draft new pitches with the template overlay.
4. Track the suggested KPIs and tune the workflow each cycle.
