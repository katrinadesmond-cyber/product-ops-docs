# Product Operations Templates (Markdown)

This repository contains Confluence product management templates converted into clean, reusable Markdown files.

The templates are designed to be:

- Cursor-friendly
- AI-tool agnostic
- GitHub-native (versionable, diffable, and easy to reuse)

## Template Library

All converted templates are in:

`templates/product-management/`

See `templates/product-management/README.md` for a full index and usage guidance.

## How to Use

1. Copy the template you need into your working docs folder.
2. Rename it for your initiative (for example, `launch-plan-q2-ads.md`).
3. Fill placeholders (typically shown in brackets like `[example]`).
4. Keep headings intact so AI tools can parse sections reliably.
5. Commit updates as you refine strategy, launch plans, and release artifacts.
