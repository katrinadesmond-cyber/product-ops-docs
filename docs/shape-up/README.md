# Shape Up AI Toolkit

This folder contains the first Product Ops efficiency initiative: improving how teams develop Shape Up pitches with AI.

## Files

- `ai-pitch-playbook.md`: Operating workflow, quality gates, and prompt starters.
- `templates/shape-up-pitch-template-ai-overlay.md`: Copy-ready template mapped to your current pitch sections.

## Recommended rollout

1. Pilot with 2-3 real pitches in the next cycle.
2. Track draft speed, revision count, and first-pass quality gate success.
3. Tune prompts and checklist based on reviewer feedback.
